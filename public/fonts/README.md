# Fonts

Drop the SN Pro variable font files here for the Film Room UI:

- `SNPro.ttf`        (upright, variable weight 100–900)
- `SNPro-Italic.ttf` (italic, variable weight 100–900)

They are referenced by `@font-face` in `app/globals.css`. Until present, the UI
falls back cleanly to the system sans (`ui-sans-serif, system-ui`).
